Installation Guide for Shourya Framework

To install the Shourya framework, follow these simple steps:

  1. use code below in your file
<script src="path/to/shourya.js"></script>
  1. Include the Shourya framework in your HTML file:
<script>
    const shouryaApp = new Shourya();
</script>

Congratulations! You have successfully installed the Shourya framework. You can now start building amazing web applications with ease.

Note: Make sure to replace path/to/shourya.js with the actual path to the Shourya framework file.
